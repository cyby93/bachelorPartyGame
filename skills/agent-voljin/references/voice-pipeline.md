# Voice Pipeline

Treat voiced dialog as authored gameplay presentation, not an afterthought.

Focus on outcomes:

- text and voice stay aligned
- placeholder lines can ship before final assets exist
- missing clips fail gracefully
- naming and timing are stable enough for asset sourcing in parallel

Read `shared/LevelConfig.js`, `server/systems/DialogSystem.js`, `shared/AudioConfig.js`, and `docs/AUDIO.md` before changing anything.
