# Asset Ingest

Turn sourced audio into a maintainable runtime library.

Focus on outcomes:

- assets have stable logical keys
- file organization is predictable
- original WoW and custom assets can coexist cleanly
- gameplay code never depends on ad hoc file paths

Read `docs/AUDIO.md` and `shared/AudioConfig.js` first. Keep logical keys stable even if source files change.
