## Session — 2026-04-17, calculator fixes + multi-target mode

**What happened:** Fixed two bugs in the HPS/DPS calculators and added multi-target (`--targets=N`) support to both. Cyby declared class balance done for now and is moving on to building UX and combat system agents.

**Key outcomes:**

- **HPS bug fixed — Chain Heal cast time ignored:** `getCastTime()` only checked for `'CAST'` and `'CHANNEL'` types. Chain Heal is `type: 'TARGETED'` with `castTime: 1500` — it was treated as instant, causing ~300 HPS inflation. Fix: return `skill.castTime ?? 0` for any skill type. Chain Heal now correctly shows 10.0 HPS (single-target).
- **DPS bug fixed — Searing Totem never cast:** Instant abilities (0 cast time) always lost the priority score race against cast-time abilities. Fix: split simulation action into two phases — Phase 1 fires all available instants (free, no blocking), Phase 2 selects best cast-time ability. Searing Totem now correctly contributes 4.8 DPS (single-target).
- **Multi-target mode added to both calculators:** `--targets=N` flag. Scales: DoT ticks ×N, AOE heals/damage ×N, SPAWN/TOTEM ×N, Chain Heal up to min(maxChains+1, N) targets, HoTs ×N. Single-target abilities unchanged.
- **Agent skill file scaffolded:** `.claude/skills/agent-balance-theorist/SKILL.md` created so Coda can be invoked via `/agent-balance-theorist` in any new context.
