# Product Overview

RAID NIGHT - THE RESCUE is a local multiplayer twin-stick MOBA party game where smartphones become controllers. Players team up to defeat raid bosses in real-time cooperative gameplay.

## Core Concept

- Host display runs on TV/monitor showing the game arena
- Each player uses their smartphone as a controller with virtual joysticks
- 8 unique classes with 4 skills each
- Real-time multiplayer combat with death/revive mechanics
- Boss fights with multiple difficulty phases

## Key Features

- Up to 13 simultaneous players
- Interactive lobby for skill practice
- Smartphone-based controller interface with touch controls
- Real-time WebSocket communication
- Modular skill system with projectiles, melee, AOE, healing, buffs, and movement abilities
- Death/revive mechanics where allies can resurrect fallen teammates
- Cooldown system with visual feedback on controllers

## Target Experience

Fast-paced party game designed for local multiplayer sessions where friends gather around a shared screen and use their phones as controllers. Emphasis on cooperative gameplay and class synergy.
