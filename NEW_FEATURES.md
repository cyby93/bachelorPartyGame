## Statistics

A feature which can record and display several statistics:
    - damage done (for each player in each level)
    - healing done by player (for each player in each level)
    - deaths of each player (for each player in each level)
    - resurrections done (for each player in each level)
    - answered quiz questions (correct/wrong)

The main goal for this feature is to display these statistics in:
    - damage and healing done in each level updating in real time
    - after each level display how much damage and healing done in that level
    - at the end of the game (either the players defeated or won the game) display everything: damage and healing done and deaths by each players and also summarise the quiz question statistics (correct/wrong answers by for each player)

The ingame damage and healing done section description:
    - I want to display the damage and healing done stats in separate sections
    - Both section would contains rows for each player and inside the rows would be the player name and the damage/heal done (left side name, right side damage). Next to the damage, we could calculate a damage per second value as well on the fly. (basically a damage/heal meter like in world of warcraft)

The level summary and endgame views can contain similar structured sections for each data (damage, heal, death, resurrections), but with the overall data in it.
